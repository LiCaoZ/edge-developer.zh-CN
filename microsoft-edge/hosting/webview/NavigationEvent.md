# NavigationEvent 对象

一个对象，表示在启动导航时激发的事件。

## 属性
    
### uri

目标的统一资源标识符（URI）。

此属性为只读。

```js
var uri = NavigationEvent.uri;
```

#### 属性值
类型： **DOMString**

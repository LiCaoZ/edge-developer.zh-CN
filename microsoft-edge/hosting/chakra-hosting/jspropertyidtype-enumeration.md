# JsPropertyIdType 枚举  

JavaScript 属性的类型枚举。  

## 语法  

```cpp
enum JsPropertyIdType;  
```  

## 成员  

### 值  

| 名称 | 描述 |  
|:--- |:--- |  
| JsPropertyIdTypeString | JavaScript 字符串属性的类型枚举 |  
| JsPropertyIdTypeSymbol | JavaScript 符号属性的类型枚举 |

# JsCreateSymbol 函数
创建 JavaScript 符号。
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateSymbol(  
   _In_ JsValueRef description,  
   _Out_ JsValueRef *result  
);  
```  
  
#### 参数  
 `description`  
 符号的字符串说明。 可以为 null。  
  
 `result`  
 新符号。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

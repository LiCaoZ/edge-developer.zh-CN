# JsIntToNumber 函数
从值创建一个数值 `int` 。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsIntToNumber(  
   _In_ int intValue,  
   _Out_ JsValueRef *value  
);  
```  
  
#### 参数  
 `intValue`  
 `int`要转换为数字值的。  
  
 `value`  
 新的数字值。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

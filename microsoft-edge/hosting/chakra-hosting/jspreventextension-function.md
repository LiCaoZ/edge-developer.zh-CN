# JsPreventExtension 函数
使对象成为非可扩展对象。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsPreventExtension(  
   _In_ JsValueRef object  
);  
```  
  
#### 参数  
 `object`  
 要使其不可扩展的对象。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

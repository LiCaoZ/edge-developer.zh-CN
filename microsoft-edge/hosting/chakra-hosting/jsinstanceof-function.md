# JsInstanceOf 函数
执行 JavaScript `instanceof` 运算符测试。  
  
## 语法  
  
```cpp  
JsInstanceOf(   
  _In_ JsValueRef object,  
  _In_ JsValueRef constructor,  
  _Out_ bool *result  
);  
  
```  
  
#### 参数  
 `object`  
 要测试的对象。  
  
 `constructor`  
 要对其进行测试的构造函数。  
  
 `result`  
 "Object instanceof 构造函数" 是否为 true。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

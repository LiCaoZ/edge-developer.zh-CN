# JsHasProperty 函数
确定对象是否具有属性。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsHasProperty(  
   _In_ JsValueRef object,  
   _In_ JsPropertyIdRef propertyId,  
   _Out_ bool *hasProperty  
);  
```  
  
#### 参数  
 `object`  
 可能包含属性的对象。  
  
 `propertyId`  
 属性的 ID。  
  
 `hasProperty`  
 对象（或原型）是否具有属性。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

# JsSetContextData 函数
设置 JsrtContext 的内部数据。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsSetContextData(  
  _In_ JsContextRef context,  
  _In_ void *data  
);  
  
```  
  
#### 参数  
 `context`  
 要将数据设置到的上下文。  
  
 `data`  
 指向要设置的数据的指针。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。  
  
## 备注

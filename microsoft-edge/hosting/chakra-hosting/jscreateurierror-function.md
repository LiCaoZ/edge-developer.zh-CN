# JsCreateURIError 函数
创建新的 JavaScript URIError error 对象。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateURIError(  
   _In_ JsValueRef message,  
   _Out_ JsValueRef *error  
);  
```  
  
#### 参数  
 `message`  
 错误对象的消息。  
  
 `error`  
 新的错误对象。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

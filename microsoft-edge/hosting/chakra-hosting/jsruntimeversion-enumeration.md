# JsRuntimeVersion 枚举
运行时的版本。  
  
## 语法  
  
```cpp  
enum JsRuntimeVersion;  
```  
  
## 成员  
  
### 值  
  
|名称|描述|  
|----------|-----------------|  
|`JsRuntimeVersion10`|通过 IE10 版本创建运行时。|  
|`JsRuntimeVersion11`|通过 IE11 版本创建运行时。|  
|`JsRuntimeVersionEdge`|在运行时在计算机上创建具有最高版本的运行时。|  
  
## 要求  
 **页眉：** jsrt

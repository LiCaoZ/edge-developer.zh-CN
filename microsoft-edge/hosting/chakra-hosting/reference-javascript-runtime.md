# 参考（JavaScript 运行时）
JavaScript 运行时（JsRT） Api 使你能够将脚本功能添加到在 Windows 上运行的桌面和服务器端应用程序。  
  
 如果你打算在你的应用程序中嵌入[ChakraCore](https://github.com/Microsoft/ChakraCore) ，请参阅[CHAKRACORE Wiki](https://aka.ms/corejsrtref) for JSRT 引用。

# JsValueRef Typedef
对 JavaScript 值的引用。  
  
## 语法  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## 备注  
 JavaScript 值是以下类型的值之一：未定义、Null、布尔值、字符串、数字或对象。  
  
## 要求  
 **页眉：** jsrt

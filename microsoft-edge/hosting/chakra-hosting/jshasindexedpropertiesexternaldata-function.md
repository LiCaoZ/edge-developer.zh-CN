# JsHasIndexedPropertiesExternalData 函数
确定对象在外部数据中是否具有索引属性。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsHasIndexedPropertiesExternalData(  
   _In_ JsValueRef object,  
   _Out_ bool* value  
);  
```  
  
#### 参数  
 `object`  
 该对象。  
  
 `value`  
 对象在外部数据中是否具有索引属性。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。  
  
## 备注  
 此 API 仅在 Microsoft Edge 模式下受支持。  
  
## 要求  
 **页眉：** jsrt

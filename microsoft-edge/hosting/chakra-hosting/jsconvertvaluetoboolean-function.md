# JsConvertValueToBoolean 函数
使用标准 JavaScript 语义将值转换为布尔值。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsConvertValueToBoolean(  
   _In_ JsValueRef value,  
   _Out_ JsValueRef *booleanValue  
);  
```  
  
#### 参数  
 `value`  
 要转换的值。  
  
 `booleanValue`  
 已转换的值。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

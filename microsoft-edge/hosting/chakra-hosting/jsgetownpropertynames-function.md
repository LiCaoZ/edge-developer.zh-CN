# JsGetOwnPropertyNames 函数
获取对象上的所有属性的列表。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsGetOwnPropertyNames(  
   _In_ JsValueRef object,  
   _Out_ JsValueRef *propertyNames  
);  
```  
  
#### 参数  
 `object`  
 要从中获取属性名称的对象。  
  
 `propertyNames`  
 属性名称数组。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

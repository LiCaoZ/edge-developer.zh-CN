# JsCreateObject 函数
创建新对象。
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateObject(  
   _Out_ JsValueRef *object  
);  
```  
  
#### 参数  
 `object`  
 新对象。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

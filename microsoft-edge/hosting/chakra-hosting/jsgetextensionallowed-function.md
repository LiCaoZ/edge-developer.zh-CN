# JsGetExtensionAllowed 函数
返回一个值，该值指示对象是否可扩展。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsGetExtensionAllowed(  
   _In_ JsValueRef object,  
   _Out_ bool *value  
);  
```  
  
#### 参数  
 `object`  
 要测试的对象。  
  
 `value`  
 对象是否可扩展。  
  
## 返回值  
 `JsNoError`如果操作成功，则为代码，否则为失败代码。

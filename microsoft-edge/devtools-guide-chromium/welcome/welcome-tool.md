# <a name="welcome-tool"></a>欢迎工具

首次 **打开** DevTools 时将打开欢迎工具。  它将显示：

*  指向开发人员文档的链接。
*  最新功能。
*  发行说明。
*  与开发人员工具Microsoft Edge联系的选项。

每次更新 Microsoft Edge 后，**欢迎**工具都会自动打开。

若要防止每次更新后显示**欢迎**工具，请清除“**欢迎**”标题下“**每次更新后打开选项卡**”旁边的复选框。

# 更改
查看你在 DevTools 诊断会话期间所做的 CSS 更改 (从 " [样式](./styles.md) " 和 " [计算](./computed.md) 的 CSS 编辑窗格") 。 与其他比较工具的约定相似，已删除的行以红色显示，添加的线条为绿色。

!["更改" 窗格](../media/elements_changes.png)

您也可以从右键单击上下文菜单中 () 将所做的更改复制到剪贴板、还原更改或跳转到其源位置。

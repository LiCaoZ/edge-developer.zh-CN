# interface IWebView2ProcessFailedEventHandler 

> [!NOTE]
> 此接口可能会在 SDK 版本0.8.355 后更改或不可用。 请参阅[参考](../../../webview2-api-reference.md)了解最新的 API 参考。

```
interface IWebView2ProcessFailedEventHandler
  : public IUnknown
```

调用方实现此接口以接收 ProcessFailed 事件。

## 摘要

 成员                        | 描述
--------------------------------|---------------------------------------------
[调用](#invoke) | 调用以向实施者提供对应事件的事件参数。

## 成员

#### 调用 

调用以向实施者提供对应事件的事件参数。

> 公共 HRESULT[调用](#invoke)（[IWebView2WebView](IWebView2WebView.md) * web 视图，[IWebView2ProcessFailedEventArgs](IWebView2ProcessFailedEventArgs.md) * 参数）

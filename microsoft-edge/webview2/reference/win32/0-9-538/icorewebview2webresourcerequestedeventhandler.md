# interface ICoreWebView2WebResourceRequestedEventHandler 

```
interface ICoreWebView2WebResourceRequestedEventHandler
  : public IUnknown
```

在 web 视图中发出 HTTP 请求时激发。

## 摘要

 成员                        | 描述
--------------------------------|---------------------------------------------
[调用](#invoke) | 调用以向实施者提供对应事件的事件参数。

主机可以替代请求、响应标头和响应内容。

## 成员

#### 调用 

调用以向实施者提供对应事件的事件参数。

# 0.9.430-接口 ICoreWebView2FocusChangedEventHandler 

> [!NOTE]
> 此接口可能会在 SDK 版本0.9.430 后更改或不可用。 请参阅[参考](../../../webview2-api-reference.md)了解最新的 API 参考。

```
interface ICoreWebView2FocusChangedEventHandler
  : public IUnknown
```

调用方实现此方法以接收 GotFocus 和 LostFocus 事件。

## 摘要

 成员                        | 描述
--------------------------------|---------------------------------------------
[调用](#invoke) | 调用以向实施者提供对应事件的事件参数。

此事件没有事件参数。

## 成员

#### 调用 

调用以向实施者提供对应事件的事件参数。

> 公共 HRESULT[调用](#invoke)（[ICoreWebView2Host](ICoreWebView2Host.md) * sender、IUnknown * 参数）

没有事件参数，args 参数将为 null。

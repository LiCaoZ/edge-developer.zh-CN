# interface ICoreWebView2ExecuteScriptCompletedHandler 

> [!NOTE]
> 此接口可能会在 SDK 版本0.9.430 后更改或不可用。 请参阅[参考](../../../webview2-api-reference.md)了解最新的 API 参考。

```
interface ICoreWebView2ExecuteScriptCompletedHandler
  : public IUnknown
```

调用方实现此接口以接收 ExecuteScript 方法的结果。

## 摘要

 成员                        | 描述
--------------------------------|---------------------------------------------
[调用](#invoke) | 调用以向实施者提供相应的异步方法调用的完成状态和结果。

#### 调用

使用 Microsoft Edge WebView2 控件，可以使用基于Chromium的Microsoft Edge作为呈现引擎在应用程序中托管 Web 内容。

若要下载 Microsoft Edge (Beta、Dev 和 Canary) 的预览频道，请转到[预览体验成员Microsoft Edge](https://www.microsoftedgeinsider.com)。

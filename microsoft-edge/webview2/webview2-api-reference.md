# <a name="microsoft-edge-webview2-api-reference"></a>Microsoft Edge WebView2 API 参考

选择要用于以下列表的 WebView2 的语言和框架。

    *   [Windows 窗体][DotnetMicrosoftWebWebView2WinformsNamespace]

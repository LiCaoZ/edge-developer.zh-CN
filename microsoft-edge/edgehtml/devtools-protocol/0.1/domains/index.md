# DevTools 协议域 - DevTools 协议版本 0.1 (EdgeHTML)   

## [调试程序](debugger.md)  

调试器域公开 JavaScript 调试功能。 它允许设置和删除断点、逐步执行、探索堆栈跟踪等。
## [页面](page.md)
与检查的页面相关的操作和事件属于页面域。
## [运行时](runtime.md)
运行时域通过远程评估和镜像对象公开 JavaScript 运行时。 评估结果作为镜像对象返回，该对象公开对象类型、字符串表示形式和唯一标识符，可用于进一步的对象引用。 原始对象将保留于内存中，除非显式释放它们。
## [架构](schema.md)
提供有关协议架构的信息。

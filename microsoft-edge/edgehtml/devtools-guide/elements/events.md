# 事件 

查看在选定元素上注册的事件侦听器， (选择使用复选框切换) 其上级。 这可用于跟踪未授权事件侦听器。 

可以按 Event 或 **Element** 对事件侦听器 **进行分组**。 单击事件处理程序名称旁边的蓝色超链接将打开调试器到该函数的位置。

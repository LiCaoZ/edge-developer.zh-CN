# 更改
从"样式"和"计算 CSS 编辑"窗格 (在 DevTools[](./styles.md)诊断[](./computed.md)会话期间所做的更改) 。 与其他差分工具的约定类似，已删除的线条显示为红色，而添加的线条是绿色。

![更改窗格](../media/elements_changes.png)

还可以从 (单击的上下文菜单中) 将更改复制到剪贴板、还原更改或跳转到其源位置。

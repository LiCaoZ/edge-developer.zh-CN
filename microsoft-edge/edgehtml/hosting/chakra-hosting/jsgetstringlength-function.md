# JsGetStringLength 函数

获取字符串值的长度。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsGetStringLength(  
   _In_ JsValueRef stringValue,  
   _Out_ int *length  
);  
```  
  
#### 参数  
 `stringValue`  
 要获取其长度的字符串值。  
  
 `length`  
 字符串的长度。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

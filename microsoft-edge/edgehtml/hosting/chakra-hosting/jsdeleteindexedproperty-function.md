# JsDeleteIndexedProperty 函数

删除对象指定索引的值。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsDeleteIndexedProperty(  
   _In_ JsValueRef object,  
   _In_ JsValueRef index  
);  
```  
  
#### 参数  
 `object`  
 要操作的对象。  
  
 `index`  
 要删除的索引。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

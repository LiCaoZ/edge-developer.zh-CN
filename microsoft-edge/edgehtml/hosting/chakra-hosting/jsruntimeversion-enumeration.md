# JsRuntimeVersion 枚举

运行时的版本。  
  
## 语法  
  
```cpp  
enum JsRuntimeVersion;  
```  
  
## 成员  
  
### 值  
  
|名称|描述|  
|----------|-----------------|  
|`JsRuntimeVersion10`|使用 IE10 版本创建运行时。|  
|`JsRuntimeVersion11`|使用 IE11 版本创建运行时。|  
|`JsRuntimeVersionEdge`|在运行时创建计算机上具有最高版本的运行时。|  
  
## 要求  
 **标头** ：jsrt.h

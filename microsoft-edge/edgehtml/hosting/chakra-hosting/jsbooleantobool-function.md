# JsBooleanToBool 函数

检索 `bool` 布尔值的值。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsBooleanToBool(  
   _In_ JsValueRef value,  
   _Out_ bool *boolValue  
);  
```  
  
#### 参数  
 `value`  
 要转换的值。  
  
 `boolValue`  
 转换后的值。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

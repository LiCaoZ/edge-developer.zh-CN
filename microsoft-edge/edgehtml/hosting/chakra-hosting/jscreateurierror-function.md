# JsCreateURIError 函数

创建新的 JavaScript URIError 错误对象。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateURIError(  
   _In_ JsValueRef message,  
   _Out_ JsValueRef *error  
);  
```  
  
#### 参数  
 `message`  
 错误对象的消息。  
  
 `error`  
 新的错误对象。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

# JsStrictEquals 函数

比较两个 JavaScript 值以严格相等。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsStrictEquals(  
   _In_ JsValueRef object1,  
   _In_ JsValueRef object2,  
   _Out_ bool *result  
);  
```  
  
#### 参数  
 `object1`  
 要比较的第一个对象。  
  
 `object2`  
 要比较的第二个对象。  
  
 `result`  
 值是否严格相等。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 此函数等效于 `===` Javascript 中的运算符。  
  
 需要活动脚本上下文。

 **标头** ：jsrt.h

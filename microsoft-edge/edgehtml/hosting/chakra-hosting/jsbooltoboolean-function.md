# JsBoolToBoolean 函数

从值创建布尔 `bool` 值。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode JsBoolToBoolean(  
   _In_ bool value,  
   _Out_ JsValueRef *booleanValue  
);  
```  
  
#### 参数  
 `value`  
 要转换的值。  
  
 `booleanValue`  
 转换后的值。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

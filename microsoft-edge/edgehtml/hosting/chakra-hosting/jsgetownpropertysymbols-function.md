# JsGetOwnPropertySymbols 函数

获取对象上所有符号属性的列表。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsGetOwnPropertySymbols(  
   _In_ JsValueRef object,  
   _Out_ JsValueRef *propertySymbols  
);  
```  
  
#### 参数  
 `object`  
 要获取其属性符号的对象。  
  
 `propertySymbols`  
 属性符号数组。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
 此 API 仅在 Microsoft Edge 模式下受支持。  
  
## 要求  
 **标头** ：jsrt.h

# JsValueRef Typedef

对 JavaScript 值的引用。  
  
## 语法  
  
```cpp 
typedef JsRef JsValueRef;  
```  
  
## 备注  
 JavaScript 值是下列值类型之一：Undefined、Null、Boolean、String、Number 或 Object。  
  
## 要求  
 **标头** ：jsrt.h

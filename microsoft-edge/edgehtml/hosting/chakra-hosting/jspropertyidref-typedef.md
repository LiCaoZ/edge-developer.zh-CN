# JsPropertyIdRef Typedef

属性标识符。  
  
## 语法  
  
```cpp  
typedef JsRef JsPropertyIdRef;  
```  
  
## 备注  
 属性标识符用于引用 JavaScript 对象的属性，而不是使用字符串。  
  
## 要求  
 **标头** ：jsrt.h

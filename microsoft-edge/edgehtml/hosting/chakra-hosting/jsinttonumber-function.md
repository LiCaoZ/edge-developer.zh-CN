# JsIntToNumber 函数

从值创建数字 `int` 值。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsIntToNumber(  
   _In_ int intValue,  
   _Out_ JsValueRef *value  
);  
```  
  
#### 参数  
 `intValue`  
 `int`要转换为数字值的。  
  
 `value`  
 新数字值。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

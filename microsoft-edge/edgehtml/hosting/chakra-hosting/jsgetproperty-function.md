# JsGetProperty 函数

获取对象的属性。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsGetProperty(  
   _In_ JsValueRef object,  
   _In_ JsPropertyIdRef propertyId,  
   _Out_ JsValueRef *value  
);  
```  
  
#### 参数  
 `object`  
 包含该属性的对象。  
  
 `propertyId`  
 属性的 ID。  
  
 `value`  
 属性的值。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

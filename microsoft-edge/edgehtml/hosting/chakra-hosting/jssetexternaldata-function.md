# JsSetExternalData 函数

设置外部对象上的外部数据。  
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsSetExternalData(  
   _In_ JsValueRef object,  
   _In_opt_ void *externalData  
);  
```  
  
#### 参数  
 `object`  
 外部对象。  
  
 `externalData`  
 对象中存储的外部数据。 如果对象中未存储任何外部数据，则其值可为空。  
  
## 返回值  
 如果 `JsNoError` 操作成功，则代码，否则为失败代码。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h

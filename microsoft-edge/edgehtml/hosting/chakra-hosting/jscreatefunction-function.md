# JsCreateFunction 函数

创建新的 JavaScript 函数。
  
## 语法  
  
```cpp  
STDAPI_(JsErrorCode) JsCreateFunction(  
   _In_ JsNativeFunction nativeFunction,  
   _In_opt_ void *callbackState,  
   _Out_ JsValueRef *function  
);  
```  
  
#### 参数  
 `nativeFunction`  
 调用函数时要调用的方法。  
  
 `callbackState`  
 将传递回回调的用户提供的状态。  
  
 `function`  
 新函数对象。  
  
## 返回值  
 呼叫的结果（如果有）。  
  
## 备注  
 需要活动脚本上下文。  
  
## 要求  
 **标头** ：jsrt.h
